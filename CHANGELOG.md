## [0.3.0] - 2026-02-24

### Added

- Data export module (`kerygma_social.data_export`) generating delivery log schema and POSSE manifest
- `social-export` CLI entry point for artifact generation
- `data/delivery-log.json` — record schema, status/platform enums, sample record
- `data/posse-manifest.json` — POSSE pattern, platforms, resilience stack, config
